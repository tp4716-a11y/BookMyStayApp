# 🏨 Book My Stay App

## 📌 Overview

**Book My Stay App** is a console-based Hotel Booking Management System built using Core Java.
This project demonstrates how fundamental programming concepts and data structures can be applied to solve real-world problems such as booking management, request handling, and inventory consistency.

The system is designed incrementally through multiple use cases, each focusing on a specific concept to build a scalable and maintainable application.

---

## 🎯 Objective

* Understand how Java applications start and execute
* Learn the importance of structured program flow
* Demonstrate core Java concepts through practical implementation
* Build a foundation for advanced features like booking logic and data handling

---

## 🚀 Use Case 1: Application Entry & Welcome Message

### 🔹 Goal

To establish a clear starting point for the application and demonstrate how a Java program executes.

### 👤 Actor

User (runs the application via command line or IDE)

### 🔄 Flow

1. User runs the application
2. JVM invokes the `main()` method
3. Application displays a welcome message
4. Application terminates

---

## 🧠 Concepts Covered

* **Class Structure**

  * Defines the program boundary

* **main() Method**

  * Entry point of execution
  * Signature: `public static void main(String[] args)`

* **static Keyword**

  * Allows method execution without object creation

* **Console Output**

  * `System.out.println()` for displaying messages

* **String Literals**

  * Immutable text stored in String pool

* **Execution Flow**

  * Sequential (top-to-bottom)

* **JavaDoc Comments**

  * Documentation for classes and methods
